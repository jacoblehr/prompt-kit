# Stack: reflect-act

Extract reusable lessons from any outcome and assign concrete follow-on actions.

Blocks:
1. `mode.reflect`
2. `frame.cause-mapping`
3. `frame.extract-insights`
4. `schema.plan-next-actions`

Expected output: Root causes identified, highest-leverage lessons named, and actions assigned so learning transfers to future work.

## Composition notes

`mode.reflect` holds space for honest evaluation before prescribing next steps. `frame.cause-mapping` traces outcomes to structural causes rather than proximate triggers. `frame.extract-insights` converts the analysis into reusable lessons. `schema.plan-next-actions` converts lessons into concrete actions with owners so learning transfers to future work.

**Minimum blocks:** `frame.cause-mapping` + `frame.extract-insights`
