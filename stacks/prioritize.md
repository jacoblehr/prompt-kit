# Stack: prioritize

Rank options under real constraints and commit to a sequenced execution order.

Blocks:
1. `mode.decide`
2. `frame.success-criteria`
3. `guardrail.assumption-audit`
4. `schema.execution-brief`

Expected output: Ranked options with explicit criteria, tradeoffs visible, and sequenced execution brief as output.

## Composition notes

`mode.decide` frames the exercise as a commitment, not an analysis. `frame.success-criteria` makes the ranking standard explicit before any options are evaluated. `guardrail.assumption-audit` surfaces hidden constraints on feasibility or sequencing. `schema.execution-brief` commits the priority order in a sequenced plan.

**Minimum blocks:** `mode.decide` + `frame.success-criteria`
