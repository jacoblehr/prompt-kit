# Stack: Weekly Review

Use when you want to learn from a week without creating a huge ritual.

Useful inputs:

- weekly notes, outcomes, or calendar fragments
- key wins, failures, or stuck points
- any major decisions worth revisiting

Suggested blocks:

1. `mode.reflect`
2. `core.weekly-review`
3. `core.cause-mapping` for key failures
4. `core.decision-journal-entry` for any major decision worth revisiting
5. `rubric.reflection-quality`

Expected outcome:

- lessons
- repeated frictions
- next-week focus
- decisions worth tracking

Domain tags:
- reflection
- personal effectiveness
- learning
