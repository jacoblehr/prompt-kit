# Stack: align-stakeholders

Map audiences and land a change or decision so it is understood and acted on, not just received.

Blocks:
1. `frame.task`
2. `mode.decide`
3. `schema.execution-brief`

Optional add-ons:
- `mode.explore` when premature convergence is a real risk.
- `guardrail.assumption-audit` when hidden premises could change the answer.

Expected output: Audience-specific communication plan with sequencing, objection handling, and a clear ask for each group.

## Composition notes

`frame.task` anchors the communication challenge before audience mapping begins. `mode.explore` opens the audience landscape before narrowing to a specific message. `guardrail.assumption-audit` tests what you believe about your audiences before committing to a plan. `mode.decide` closes to a sequenced approach. `schema.execution-brief` shapes delivery into a concrete, assignable output.

**Common failure mode:** Treating alignment as one message for everyone instead of a sequenced set of asks by audience.

**Minimum blocks:** `frame.task` + `mode.decide` + `schema.execution-brief`
