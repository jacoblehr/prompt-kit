# Domain: data

Context overlay for data analysis, statistics, and research tasks.

- Correlation is not causation; name the confound before recommending action.
- Statistical significance does not imply practical significance — report effect size.
- Survivorship bias: ask what data is absent before concluding from what is present.
- Base rates matter: surprising findings are more likely to be measurement errors than true signals.
- Pre-register the hypothesis before looking at the data whenever feasible.
- Confidence intervals, not just p-values; state what the range of plausible effects is.
