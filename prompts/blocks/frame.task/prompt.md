# Prompt

Restate this raw request or situation as a structured problem frame.

Requirements:

- separate the stated ask from the likely objective
- state when the objective is inferred rather than explicit
- keep constraints literal
- surface unknowns without pretending to resolve them
- recommend the next reasoning move rather than a full solution

Return:

- stated ask
- likely objective
- constraints
- knowns
- unknowns
- decision or action this frame is meant to support
- recommended next reasoning step

Input:
{paste raw request or situation}
