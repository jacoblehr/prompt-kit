# Prompt

Render this hypothesis or uncertain claim as an experiment plan.

Requirements:

- state the decision the test is meant to inform
- define the method before interpreting evidence
- choose metrics or observations that can actually answer the question
- name confounds, sampling limits, and operational risks
- include a decision rule before the result is known

Return:

- hypothesis
- decision this test informs
- method
- metric or observation
- sample or scope
- confounds and risks
- decision rule
- next action after each outcome
