# frame.cause-mapping

## Purpose
Map cause-effect chains behind an outcome; separate contributing factors from root cause.

## Use when
An outcome needs explanation beyond surface description. The main risk is misidentifying a symptom as the cause.

## Expects
A problem, failure, or outcome to trace.

## Adds
A multi-level causal chain with branching points and the earliest feasible intervention identified.

## Returns
- visible symptom
- causal chain (≥ 3 levels)
- branch points
- earliest intervention point
- most likely root cause

## Pairs with
`mode.reflect`, `mode.critique`, `frame.extract-insights`, `schema.incident-postmortem`

## Avoid when
The outcome is still unfolding and causal chains cannot yet be traced. Use `guardrail.uncertainty` first.
