# Statistical Significance Check

Use when interpreting results from an experiment, A/B test, or data analysis to avoid acting on noise.

```text
Interpret these test results with statistical rigour.

Check:
1. Sample size: was the sample large enough to detect the effect size that would actually matter for a decision?
2. Pre-registration: was the hypothesis and primary metric set before data collection, or post-hoc?
3. Statistical significance: what is the p-value and confidence interval on the primary metric?
4. Practical significance: is the effect size large enough to matter, even if statistically significant?
5. Multiple comparisons: were many metrics or segments tested? If so, inflate the significance threshold accordingly.
6. Novelty effects: could the result reflect short-term response to change rather than true steady-state behaviour?
7. Segment breakdown: does the effect hold across key sub-groups, or is it driven by one segment?

Verdict: should this result be acted on, replicated, or discarded — and why?

Experiment or test results:
{paste result summary: metric, p-value, confidence interval, sample size, duration}
```
