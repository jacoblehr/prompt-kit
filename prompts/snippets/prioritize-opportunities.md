# Prioritize Opportunities

Use when you have too many plausible projects, bets, or tasks competing for attention.

```text
Prioritize these opportunities or tasks.

Return columns:
- item
- expected upside
- strategic fit
- effort or cost
- urgency or timing
- reversibility
- key risk
- recommendation (do now / later / drop)

Requirements:
- explain the top two picks
- identify one attractive option that should still wait
- avoid ranking purely by excitement

Context:
{paste context}

Items:
{paste items}
```
