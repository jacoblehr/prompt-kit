# Experiment Design

Use when you need to design a rigorous experiment or A/B test before running it.

```text
Design an experiment to test the following hypothesis.

Hypothesis: {hypothesis}

Context: {context}

Specify:
1. Treatment and control — what exactly is being compared, and what the baseline is
2. Primary metric — the one measurable outcome that determines success or failure
3. Secondary metrics — supporting signals and guardrail metrics
4. Sample requirements — who must be included, and the reasoning behind minimum sample size
5. Duration — how long to run the experiment and why
6. Risks and confounds — what could invalidate results or introduce bias
7. Decision rule — what outcome triggers each possible action

Be specific. Flag any assumptions embedded in the design.
```
