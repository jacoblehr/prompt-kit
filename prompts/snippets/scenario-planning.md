# Scenario Planning

Use when a decision depends on how the future unfolds and the range of plausible futures is wide.

```text
Generate three to four distinct future scenarios for this situation.

Requirements:
- scenarios should differ on the key uncertainties, not just optimism
- each should be internally consistent and plausible
- name the driving forces that distinguish each scenario
- for each scenario: name it, describe the world it implies, and note what would be different about the right strategy in that world
- identify which scenario would be the worst to be unprepared for
- identify the earliest observable signal that each scenario is forming

Situation or decision:
{paste situation or decision}

Key uncertainties:
{paste main unknowns driving the range of outcomes}
```

Domain tags:
- strategic thinking
- foresight
- risk management
