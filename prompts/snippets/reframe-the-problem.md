# Reframe the Problem

Use when the stated problem may be the wrong problem, or when solutions keep failing.

```text
Challenge the framing of this problem and offer better alternatives.

Requirements:
- state the problem as currently framed
- identify the hidden assumption baked into that framing
- generate three alternative framings that dissolve or transform the problem
- for each: state what changes if you accept this frame instead
- recommend which frame is most likely to lead to a tractable solution
- name what would need to be true for the original framing to be correct

Problem as stated:
{paste problem statement}

What has already been tried:
{paste any prior attempts or constraints}
```
