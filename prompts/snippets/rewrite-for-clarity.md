# Rewrite For Clarity

Use when a draft is muddled, bloated, or indirect.

```text
Rewrite this for clarity.

Requirements:
- keep the meaning
- remove redundancy
- tighten structure
- make the key point obvious earlier
- preserve useful nuance

Text:
{paste draft}
```
