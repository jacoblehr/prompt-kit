# Synthesize Sources

Use when you have consumed multiple sources and need to integrate them into a coherent, unified view rather than a list of summaries.

```text
Synthesize these sources into a unified analysis.

Steps:
- identify the core question or problem all the sources address
- map the points of agreement across sources
- map the points of genuine disagreement, noting what drives each difference
  (different evidence, different values, different framing, or different time period)
- identify any significant gap: the important point that none of the sources address
- produce an integrated position that honestly reflects the weight of evidence,
  explicitly noting remaining uncertainty

Format as a synthesis, not as a sequence of summaries. Cite the source when
crediting a specific claim.

Sources:
{paste or summarize the key content from each source, labelled Source 1, Source 2, etc.}

Central question:
{paste the question or problem you are trying to resolve}
```
