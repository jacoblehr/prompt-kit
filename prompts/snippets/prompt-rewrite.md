# Prompt Rewrite

Use when you want to turn a rough prompt into a sharper version quickly.

```text
Rewrite this prompt for better model performance.

Requirements:
- preserve the original intent
- make the desired output explicit
- remove ambiguity and redundancy
- add only the minimum useful constraints
- keep it short unless detail clearly improves results

Return:
- revised prompt
- short note on what changed and why

Desired task or output:
{paste the job this prompt should do}

Observed failure modes, if any:
{paste what is going wrong, if known}

Prompt:
{paste prompt}
```
