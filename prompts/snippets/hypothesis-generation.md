# Hypothesis Generation

Use when you have a question or problem and need to generate competing explanations before committing to an investigation path.

```text
Generate a set of competing hypotheses to explain or address this.

Requirements:
- produce at least 4–6 distinct hypotheses, not variations of the same idea
- span different levels of explanation (proximate, structural, behavioural)
- state each as a falsifiable claim: if X is true, we would expect Y
- briefly note what evidence would confirm or disconfirm each
- rank them roughly by prior plausibility given what is already known
- flag any hypothesis that is uncomfortable but worth taking seriously

Question or problem:
{paste question, anomaly, or problem}

Known evidence or context:
{paste what is already known}
```
