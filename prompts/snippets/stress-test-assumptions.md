# Stress Test Assumptions

Use when a plan or decision rests on assumptions that have not been pressure-tested.

```text
Stress test the assumptions behind this.

For each significant assumption:
- state the assumption
- rate confidence (high / medium / low)
- name the failure mode if the assumption is wrong
- suggest the cheapest way to verify it before committing

Focus first on assumptions where being wrong would most damage the outcome.

Input:
{paste plan, decision, or argument}
```
