# Define Success Metrics

Use when a goal, task, or initiative needs clearer measurement criteria before work begins.

```text
Define the success metrics for this goal or initiative.

For each dimension of success:
- name it clearly
- describe what "good" looks like at completion (not during the process)
- suggest the most direct way to measure it
- identify a leading indicator that would signal early progress
- flag any metrics that could be gamed or that measure the wrong thing

Distinguish between metrics that measure output (activity) and ones that measure
outcome (actual impact).

Then identify: which single metric, if it moved, would most definitively confirm
that the goal was achieved?

Goal or initiative:
{paste goal, task, or project description}
```
