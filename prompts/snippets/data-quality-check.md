# Data Quality Check

Use when you need to assess whether a dataset is fit for a specific analytical purpose.

```text
Assess whether this data is fit for its intended purpose.

Dataset description: {dataset_description}

Intended use: {intended_use}

Evaluate:
1. Completeness — what is missing or sparse, and whether it materially affects the intended use
2. Accuracy — what might be mislabelled, inconsistently measured, or incorrect
3. Representativeness — whether the sample reflects the population of interest
4. Recency — whether the data is current enough; what may have changed since collection
5. Bias risks — systematic distortions that could skew conclusions
6. Fitness verdict — overall suitability for the intended use, and caveats that must accompany any findings

Name the single biggest data quality concern prominently at the top.
```
