# Generate Options

Use when you need multiple directions before choosing.

```text
Generate a diverse set of plausible options.

Requirements:
- include genuinely different approaches, not small variations
- include at least one option that contradicts the default instinct
- note what each option optimizes for
- note the main downside of each option
- surface any obvious unknowns
- name the comparison dimension that matters most next

Return:
- option
- what it optimizes for
- downside
- unknowns
- next comparison angle

Problem:
{paste problem}

Constraints, limits, or non-goals:
{paste constraints, limits, or non-goals}
```
