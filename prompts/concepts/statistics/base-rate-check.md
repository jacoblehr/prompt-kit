# Base Rate Check

Apply when estimating probabilities or making predictions, to avoid neglecting the historical frequency of similar cases.

```text
Apply base rate thinking to this forecast or estimate.

Steps:
- identify the reference class: what category of similar cases does this belong to?
- find or estimate the base rate for that class (historical frequency of the outcome)
- note how confident you are in the reference class — is it genuinely comparable?
- adjust from the base rate only for specific, clearly relevant factors (not vague optimism)
- state your final calibrated estimate with explicit reasoning for any departure from the base rate
- identify where overconfidence or underconfidence is most likely to distort this estimate

Forecast or estimate:
{paste claim, prediction, or probability assessment}
```
