# Confidence Calibration

Apply when a plan, argument, or claim rests on confident assertions that have not been examined for whether the expressed certainty is warranted.

```text
Audit the confidence levels in this work for calibration.

For each major claim or estimate:
- state the current implied confidence level (high / medium / low or a rough probability)
- identify what would need to be true for that confidence level to be justified
- flag claims where confidence exceeds the available evidence
- flag claims where false modesty is blocking useful commitment
- suggest the well-calibrated version of each problematic claim

Then give an overall verdict: overconfident, underconfident, or well-calibrated —
and name the single claim whose confidence most deserves scrutiny.

Work to audit:
{paste plan, argument, forecast, or set of claims}
```
