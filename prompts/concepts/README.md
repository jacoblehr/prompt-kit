# Lenses

These prompts are organized by reusable conceptual lens rather than by task type.

In the simplified taxonomy, they are `lens` blocks.

Use them when a game theory, psychology, or computer science frame would make the problem easier to reason about.

They work best when paired with a mode block and, if needed, a simple guardrail block.

Included lenses:

- game theory
- psychology
- computer science
