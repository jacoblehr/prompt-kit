# Philosophy: First Principles

Use when a problem or solution rests on inherited assumptions that should be examined from scratch.

```text
Break this down to first principles.

Requirements:
- identify what is assumed to be true without direct justification
- strip each assumption back to the most fundamental claim that can still be verified
- list what remains when all inherited assumptions are removed
- from those fundamentals, re-derive the approach without retracing the same path
- state what genuinely changes if you build back up from scratch

Problem or assumption set:
{paste the problem, system, approach, or belief to examine}
```
